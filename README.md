# Todo Application

A full-stack Todo application with Spring Boot backend and Flutter frontend.

## Backend Setup

1. Prerequisites:
   - Java 11+
   - MySQL
   - Maven

2. Database Setup:
   - Create a MySQL database named `todo_db`
   - Update `application.properties` with your database credentials

3. Run the application:
   ```bash
   mvn spring-boot:run
